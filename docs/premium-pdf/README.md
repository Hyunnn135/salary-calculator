## 1. 왜 지금 PDF 콘텐츠인가

애드센스 승인 대기 기간(수일~수주)은 코드 작업보다 **수익화 구조 설계**에 투자할 가치가 있는 시간이다. 월급연구소는 106개 계산기 페이지로 "계산 도구" 포지셔닝을 이미 확보했으므로, 다음 단계는 **계산 결과를 해석·활용·절세로 연결시켜주는 심화 정보 상품**이다. PDF는 이 틈을 가장 낮은 제작 비용으로 메울 수 있는 포맷이다.

**PDF의 전략적 역할 3가지:**

1. **유료 수익원** (단일 결제 9,900~19,900원 / 번들 29,900원)
2. **리드매그넷** (이메일 수집 → 뉴스레터 → 재방문 유도)
3. **사이트 권위 신호** (전문성 과시 → 애드센스 승인 심사에도 긍정적, 제휴 마케팅 광고주 설득력 상승)

수익 기대치(보수적 추정)는 일 방문자 500명 · 전환율 0.3% · 평균 ASP 12,000원 기준 **월 54만 원** 수준이며, 단순 계산기 사이트가 월 10~30만 원 AdSense를 목표하는 것과 비교해 규모 있는 기여를 기대할 수 있다.

- **03은 프리미엄** — 검색 수요는 좁지만 구매 의지 강함, 높은 마진

## 4. 가격 전략

**왜 14,900 / 19,900원인가:**
- "2만 원 이하 = 결제 저항 낮음"이라는 일반 휴리스틱
- 책 한 권 값 프레임(교보문고 실용서 평균 16,000원대)을 의식적으로 따라감
- 9,900원은 "너무 저렴 = 가치 없음" 인식을 초래할 수 있어 회피
- 세트 할인은 "묶음 구매 유도 + 객단가 상승" 동시 효과

**결제 옵션:**
- 카카오페이 / 네이버페이 / 카드 (일반 간편결제 기본)
- 원클릭 결제 후 즉시 다운로드 링크 제공

**환불 정책:**
- 결제 후 24시간 내 다운로드 전일 경우 전액 환불
- 다운로드 후에는 환불 불가 (콘텐츠 복제 가능성 고려)

## 5. 배포·결제 플랫폼 비교

| 플랫폼 | 수수료 | 장점 | 단점 | 적합도 |
|---|---|---|---|---|
| **크몽/탈잉** | 15~20% | 자체 트래픽, 결제 인프라 완비 | 수수료 높음, 플랫폼 종속 | 낮음 |
| **스티비** | 월 정기 구독 비용 | 뉴스레터+결제 통합 | PDF 1회 판매엔 과설계 | 리드매그넷 한정 중간 |
| **부트페이/나이스페이** | 3.3~3.5% | 카드+간편결제 전체 지원 | 통합 개발 필요 | 중간 |
| **Gumroad** | 10% + 결제 수수료 | 전세계 판매, 즉시 개설 | 원화 결제 UX 열악, 국내 구매자 거부감 | 낮음 |
| **자체 구축 (Toss Payments / PortOne)** | 2.9% | 수수료 최저, 브랜딩 100% | 개발 공수 1~2주 | **장기적으로 추천** |

**단계별 권장:**
- **Phase 1 (1~2개월):** 스티비로 02 리드매그넷 먼저 운영 + 01/03은 구글폼+수동 결제 안내로 빠르게 검증
- **Phase 2 (3개월~):** 매출이 월 50만 원 이상 확인되면 PortOne(구 아임포트) 결제 붙여서 자체 판매 페이지로 전환

## 6. 애드센스와의 공존 전략

**핵심 원칙:** PDF 판매 페이지에 애드센스 광고를 같이 노출시키지 않는다. 광고와 결제 버튼이 한 화면에 있으면 전환율이 급락(전환 동선이 분산됨).

**동선 설계:**
```
[계산기 페이지 (애드센스 O, PDF 배너 소형 O)]
         ↓ (관심 유도)
[PDF 소개/랜딩 페이지 (애드센스 X, 결제 CTA 단독)]
         ↓ (구매)
[다운로드 페이지 (애드센스 X, 업셀/관련 콘텐츠 배너 O)]
```

**각 계산기 페이지에 PDF 배너 삽입 지점:**
- 계산 결과 박스 바로 아래 (가장 관심도 높은 순간)
- 하단 "관련 콘텐츠" 섹션에 PDF 표지 썸네일 포함
- 이미 `common.css`의 `article-card` 컴포넌트로 구현 가능 — 추가 디자인 공수 최소

## 9. KPI (성과 지표)

**리드매그넷(02):**
- 이메일 수집 수: 월 100명 (Phase 1) → 300명 (Phase 2)
- 가입→구매 전환율: 3~5%

**유료 제품(01, 03):**
- 월 판매 건수: 01은 30건 / 03은 10건 (Phase 2 목표)
- 월 매출: 01 = 447,000원 / 03 = 199,000원 → **합계 약 65만 원**
- 환불율: 5% 미만

**간접 효과:**
- 재방문율(Google Analytics): +15%p 이상
- 사이트 평균 체류 시간: +30초 이상

## 10. 리스크와 대응

| 리스크 | 대응 |
|---|---|
| PDF 내용이 "블로그에서 무료로 볼 수 있는 정보"와 차별화되지 않음 | 각 PDF에 **독점 자료**(맞춤 체크리스트, 인쇄용 워크시트, 엑셀 템플릿) 최소 3종씩 포함 |
| 유입은 있는데 구매가 안 됨 | 리드매그넷(02)에서 "01 할인 쿠폰" 이메일 발송해 재공략 |
| 세법/정책 변경으로 콘텐츠 수명 단축 | "2026년 개정 세법 반영" 연도 표기 + 매년 업데이트판 출시 (리피트 구매 유도) |
| 불법 복제·공유 | 구매자 이메일 워터마크 삽입, 적발 시 법적 대응 문구 명시 |
| 결제 분쟁 | 상세한 환불 정책 + 다운로드 전 1회 미리보기 챕터 제공 |

## 11. 다음 액션

현태님 확인 & 승인이 필요한 포인트:

1. **3종 제품 라인업 방향성 OK?** — 01(연말정산)·02(체크리스트)·03(연봉 협상) 조합에 대한 의견
2. **가격대 14,900 / 19,900 OK?** 또는 더 낮춰서 9,900 / 14,900으로?
3. **배포 플랫폼 Phase 1 = 스티비 + 구글폼 방식 동의?** — 결제 솔루션 바로 붙일지, 일단 소량 판매로 검증할지
4. **집필 시작 순서** — 02(가벼움) → 01(플래그십) → 03(프리미엄) 순이 무난한데, 연말정산 시즌(11월~) 전 출시가 목표이면 01을 먼저 당길 수도 있음

이 중 확정되는 대로 각 PDF의 본문 집필(챕터별 원고 작성)에 착수한다.
