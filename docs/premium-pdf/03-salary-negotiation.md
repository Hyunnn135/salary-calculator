# PDF #3 — 연봉 협상 실무 가이드: 실수령액 기준 연봉 역산법

> **카피:** "세전 5천과 세전 6천, 손에 들어오는 돈은 왜 70만 원 차이뿐인가"
> **포지셔닝:** 프리미엄 유료 제품
> **가격:** 19,900원
> **분량:** A4 30~35p
> **출시 목표일:** 2026년 1~2월 (이직 시즌)

---

## 1. 한 줄 요약

세전 연봉이 아닌 **"실수령액 기준"으로 연봉을 역산·협상하는 법**을 다룬 실무 매뉴얼. 협상 테이블에서 바로 쓸 수 있는 스크립트와 계산 템플릿 포함.

## 2. 타겟 독자 페르소나

**주 타겟:**
- 이직을 준비 중인 3~10년차 직장인 (가장 높은 구매 동기)
- 연봉 협상 시즌(1월)에 재계약을 앞둔 직장인
- 스타트업에서 대기업으로, 또는 역방향 이동을 고려하는 사람

**부 타겟:**
- 인사·HR 담당자 (협상 상대 입장에서 참고용)
- 프리랜서로 전환하며 단가 책정이 필요한 사람

## 3. 차별화 포인트

연봉 협상 콘텐츠는 유튜브·블로그에 많지만 대부분 **"당당하게 말해라" 수준의 소프트 스킬**에 그친다. 이 PDF의 차별점:

1. **실수령액 중심 사고** — 세전 연봉 협상이 아닌 "월 실수령 X원을 맞추려면 세전 Y원"으로 역산. 이게 협상의 주도권을 가져오는 핵심.
2. **총보상(Total Compensation) 계산기** — 기본급 + 성과급 + 스톡옵션 + 식대 + 복지포인트 + 퇴직연금 기여 등 모든 보상을 "실수령액 환산가"로 합산하는 템플릿 제공.
3. **세전 vs 세후 협상 시나리오 비교표** — 똑같은 세전 6,000만 원이어도 비과세 항목 배분에 따라 실수령액이 달라지는 것을 시뮬레이션.
4. **협상 스크립트 10선** — 실제 협상에서 쓸 수 있는 문장 템플릿 (예: "제시하신 5,800만 원은 현재 연봉 대비 월 실수령 기준 X원 인상 효과인데, 제 예상치는 Y원입니다").
5. **블라인드 제안 받는 법** — 헤드헌터·회사가 먼저 연봉을 말하게 만드는 기법.
6. **미공개 보상 발굴** — 연봉표에 없는 조건(사이닝 보너스, 주식 Vesting, 재택 근무 지원금 등) 체크리스트.

## 4. 상세 목차

### Part 1. 마인드셋 — 협상의 기본 원리 (4p)
- 01장. 왜 "세전 연봉"으로 협상하면 손해 보는가
- 02장. 협상은 정보 전쟁이다 — 내가 가진 카드 / 상대가 가진 카드
- 03장. 3가지 함정: 앵커링, 블라인드 제안, 시간 압박

### Part 2. 실수령액 역산의 수학 (8p)
- 04장. 국민연금·건강보험·고용보험 + 소득세 계산 구조
- 05장. **실수령액 → 세전 연봉 역산 공식**
- 06장. 비과세 항목 활용법 (식대 20만·자가운전 20만·연구활동비 등)
- 07장. 성과급·상여금이 실수령에 미치는 영향
- 08장. 스톡옵션/RSU의 세금 처리 — 부여·행사·매각 3단계

### Part 3. 총보상 환산 (Total Compensation) (6p)
- 09장. 기본급 vs 총보상의 차이
- 10장. 복지 항목 환산표 (식대/보육비/주택지원/교육비 등을 금액으로)
- 11장. 장기 보상 계산 (퇴직금, DC형/DB형 퇴직연금, 사이닝 보너스)
- 12장. A회사 세전 6,000 vs B회사 세전 5,800 + 스톡옵션 — 실제 비교 사례

### Part 4. 협상 전략과 스크립트 (10p)
- 13장. 협상 전 준비 체크리스트 (시장가 조사, 제안서 연구, BATNA 설정)
- 14장. 채용 공고 해독법 — "연봉 협의"의 진짜 의미
- 15장. 헤드헌터·인사팀과의 첫 통화 스크립트
- 16장. 오퍼 레터 받은 후 24시간 룰 (즉시 수락의 위험)
- 17장. **반론 스크립트 10선** (각 상황별 구체 문장)
- 18장. 서명 직전 마지막 체크 항목 15가지

### Part 5. 특수 상황 (5p)
- 19장. 스타트업 이직 — 스톡옵션 가치 평가법
- 20장. 대기업 → 스타트업 / 스타트업 → 대기업 시 세금·퇴직금 주의점
- 21장. 외국계 기업 — 달러 연봉, 환율 리스크
- 22장. 프리랜서 단가 책정 — 연봉 환산법 (3.3% 원천징수 고려)

### 부록 (2p)
- 협상 상황별 스크립트 요약 1p
- salarykorea 실수령액 계산기 + 세금 시뮬레이터 QR
- 연봉 정보 공개 사이트 정리 (크레딧잡·잡플래닛·블라인드·리멤버 커리어)

## 6. 차별화 킬러 자료 — **총보상 엑셀 템플릿**

구매자 전원에게 제공하는 **독점 부록**:
- 엑셀 시트 1개 + 구글 시트 사본 템플릿
- 기본 정보 입력만 하면 자동으로:
  - 월 실수령액 예상
  - 연간 총보상 (복지·스톡 포함)
  - 동일 세후 소득을 위한 다른 회사 제안 역산
- 이게 "PDF 사는 이유"의 핵심 훅 — 돈 받는 도구
