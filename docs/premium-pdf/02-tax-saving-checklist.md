# PDF #2 — 직장인 세금 절약 체크리스트 25

> **카피:** "출근길 10분에 읽고, 연 100만 원 더 돌려받는 법"
> **포지셔닝:** 리드매그넷 (무료 · 이메일 교환)
> **가격:** 무료
> **분량:** A4 10~12p
> **출시 목표일:** 2026년 5월 초 (가장 먼저 출시 — 퍼널 구축 시작)

---

## 1. 한 줄 요약

직장인이 1년 동안 챙기면 세금을 줄이거나 환급받을 수 있는 **실행 가능한 체크 25개**를 한눈에 보는 얇은 가이드.

## 2. 포지셔닝 근거 — 왜 무료인가

**유료로 팔지 않고 무료 배포하는 이유:**
- 월급연구소의 **퍼널 입구** 역할이 본업. 이메일 주소가 진짜 상품.
- 내용이 "실행 체크 25개" 수준이라 PDF #1의 연말정산 가이드와 겹치지 않고 오히려 **맛보기 역할** — 한 줄만 깊게 풀어낸 게 PDF #1.
- 바이럴 가능성이 높음 (지인에게 공유하기 쉬운 분량). 공유될수록 이메일 DB가 복리 성장.

## 3. 타겟 독자 페르소나

**주 타겟:**
- 세금에 관심 있지만 "어디서 시작해야 할지 모르는" 2030 직장인
- 연말정산을 수동적으로만 해왔던 모든 직장인
- 뉴스레터 구독에 익숙한 얼리어답터 성향

**유입 경로:**
- 네이버/구글 검색 "직장인 세금 절약"
- salarykorea.site 각 계산기 페이지 사이드 배너
- 네이버 블로그 포스트 하단 링크
- SNS 공유 (카톡 단톡방, 트위터/스레드)

## 4. 콘텐츠 구성 — 25개 체크 항목

### Part A. 상시 (매월·매주 해야 할 것) · 8개
1. 신용카드 vs 체크카드 vs 현금 **최적 사용 비율** 유지 (총급여 25% 초과분부터 공제)
2. 대중교통·전통시장 결제 분리 (공제율 40%)
3. 문화비(도서/공연) 체크카드로 결제 (총급여 7,000만 이하)
4. 월세 자동이체 계약서·영수증 보관 (무주택 총급여 7,000만 이하)
5. 기부금 영수증 발급기관 확인 (법정/지정 구분)
6. 의료비 결제 시 본인 명의 카드 사용
7. 청약통장 월 20만 원 이하로 납입 (공제 한도 300만)
8. 출퇴근 거리 교통비 증빙 — 회사 비과세 지원 확인

### Part B. 분기별 (3개월 주기 점검) · 5개
9. 연금저축·IRP 납입액 중간 점검 (연 900만 한도)
10. 건강보험료 직장 보수 외 소득 2,000만 초과 여부
11. 주택청약 통장 납입 누락 방지
12. 자녀 교육비 영수증 수집 (초·중·고·대학)
13. 기부금 누적 확인 + 추가 기부 계획

### Part C. 연 1회 (연말정산 시즌) · 8개
14. 홈택스 연말정산 간소화 서비스 자료 전체 출력
15. 의료비 누락분 수기 제출 (간소화에 안 잡히는 한의원·안경점 등)
16. 월세 세액공제 — 계약서 + 이체 내역 함께 제출
17. 중소기업 감면 대상 확인 (사회초년생·청년)
18. 주택담보대출 이자 상환액 — 취급 은행 증명서 발급
19. 기부금 — 홈택스 누락 시 영수증 스캔 제출
20. 맞벌이 배우자와 공제 배분 시뮬레이션
21. 부양가족 공제 재확인 (소득·나이·생계)

### Part D. 이벤트 발생 시 (상황별) · 4개
22. 이직·퇴사 — 전 직장 원천징수영수증 수령
23. 결혼 — 배우자 공제 등록, 주소 이전
24. 출산 — 자녀 공제 추가 (출생연도 기준)
25. 주택 매매 — 취득세·양도세 별도 체크 (부동산연구소 참조)

각 항목마다:
- ✅ "체크 포인트" (1~2줄)
- 💰 "얼마나 아낄 수 있나" (개략 금액 또는 %)
- 🔗 관련 salarykorea 계산기 QR 코드

## 5. 제작 공수 추정

| 단계 | 예상 기간 |
|---|---|
| 체크 25개 최종 확정 + 자료 수집 | 2일 |
| 본문 집필 (10~12p) | 3~4일 |
| 디자인 (템플릿 기반, Canva로 빠르게) | 2일 |
| 교정 | 1일 |
| **합계** | **약 1주** |

가장 가벼운 제작. **1주 안에 출시 가능**한 퀄리티 목표.

## 6. 랜딩 페이지 요소

- 히어로: "연 100만 원 더 돌려받는 체크리스트 25"
- 서브: "이메일 입력 3초 → PDF 즉시 다운로드"

- 하단 "미리보기 3개 항목" (체크 1, 5, 15번 샘플 이미지)
- 사회적 증거: "이미 XXX명이 받아 갔습니다" 카운터 (초기엔 수동 업데이트)
- FAQ 3개 (스팸 안 보내는지 / 환급 보장인지 / 세무사 대체인지)

## 7. 이메일 자동화 시퀀스

스티비(Stibee)로 5단계 웰컴 시퀀스 구축:

| 발송 시점 | 콘텐츠 | 목적 |
|---|---|---|
| 즉시 | "체크리스트 다운로드 링크 + 첫 인사" | 전달 |
| D+2 | "25개 중 오늘 당장 할 수 있는 3개" 요약 | 재참여 |
| D+5 | "연말정산 시즌이 오면 절대 놓치지 말 것 TOP 5" | 가치 제공 |
| D+10 | salarykorea 인기 계산기 소개 + 블로그 포스트 추천 | 사이트 재방문 유도 |
| D+14 | "더 깊이 공부하려면?" → PDF #1(연말정산 가이드) 런칭 알림·할인 쿠폰 | **전환 CTA** |

이 14일 시퀀스로 "이메일 구독 → PDF #1 구매"의 자동 퍼널이 완성됨.
