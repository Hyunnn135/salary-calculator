# PDF #1 — 2026년 연말정산 완벽 가이드

> **카피:** "13월의 월급, 이번엔 제대로 받자"
> **포지셔닝:** 플래그십 유료 제품
> **가격:** 14,900원
> **분량:** A4 40~50p
> **출시 목표일:** 2026년 11월 초 (연말정산 시즌 직전)

---

## 1. 한 줄 요약

직장인이 연말정산에서 놓치기 쉬운 공제 항목 40여 가지를 **항목별 체크리스트 + 계산 예시 + 실무 서류 체크**로 정리한 실용서.

## 2. 타겟 독자 페르소나

**주 타겟 (70%):**
- 30~45세 직장인, 연봉 3,500만~8,000만 원
- 연말정산을 "회사에서 시키는 대로" 해왔지만 환급액이 기대보다 적어 답답함을 느낀 사람
- 세무사 상담(20~30만 원)까지는 부담스러워하는 중간층

**부 타겟 (30%):**
- 첫 연말정산을 앞둔 사회초년생(만 2~3년차)
- 맞벌이 부부 (공제 배분 전략이 필요)
- 이직·퇴사 경험자 (중도입사/퇴사 시 연말정산 특이사항)

## 3. 차별화 포인트 — "왜 이걸 돈 내고 사는가"

무료 블로그 글이 넘쳐나는데도 팔릴 이유:

1. **통합 흐름도** — 흩어진 공제 정보를 "내 상황 체크 → 관련 공제 → 필요 서류 → 예상 환급액" 한 흐름으로 재구성. 무료 콘텐츠는 단편적이라 직접 엮어 읽어야 함.
2. **연령·가족 구성별 공제 매트릭스** — 40여 항목을 "미혼 사회초년생 / 기혼 맞벌이 / 기혼 외벌이 + 자녀 / 1인 가구 중장년" 4개 페르소나별로 "해당 여부 / 평균 환급 기여액" 표로 정리. 독자가 자기 페르소나만 빠르게 훑으면 됨.
3. **인쇄용 워크시트** — 실제로 인쇄해서 영수증/증빙 서류를 붙여가며 쓸 수 있는 4쪽 워크시트 수록
4. **2026년 개정 세법 반영** — 매년 바뀌는 공제 한도, 세율, 신설/폐지 항목을 별도 컬러 박스로 강조. 작년 자료와 혼동 방지.
5. **salarykorea 계산기 바로가기 QR** — 각 챕터 말미에 해당 계산기로 연결되는 QR 코드 (예: "의료비 세액공제 챕터 → 실수령액 계산기 + 연말정산 계산기 QR")

## 4. 상세 목차 (가안)

### Part 1. 연말정산의 구조 이해 (6p)
- 01장. 연말정산이란 무엇인가 — 원천징수와 정산의 개념
- 02장. 환급 vs 추가 납부가 갈리는 이유
- 03장. 내 원천징수영수증 읽는 법 (실제 예시 도표 포함)
- 04장. 2026년 달라진 점 요약 (1p 인포그래픽)

### Part 2. 소득공제 — 근로소득에서 빼주는 항목 (12p)
- 05장. 인적공제 (본인·배우자·부양가족)
- 06장. 연금보험료·건강보험료·고용보험료
- 07장. 주택청약종합저축, 주택임차차입금
- 08장. 신용카드·체크카드·현금영수증 소득공제 — **최적 사용 비율 공식**
- 09장. 중소기업 취업자 감면 (사회초년생 필독)

### Part 3. 세액공제 — 계산된 세금에서 빼주는 항목 (14p)
- 10장. 자녀 세액공제 및 출산·입양 공제
- 11장. 연금계좌 세액공제 (연금저축 + IRP 한도 전략)
- 12장. 보장성 보험료 세액공제
- 13장. 의료비 세액공제 — 놓치기 쉬운 포인트
- 14장. 교육비 세액공제 (본인 + 부양가족)
- 15장. 기부금 세액공제 (법정/지정/정치/종교 구분)
- 16장. 월세액 세액공제 (무주택 총급여 7천 이하)
- 17장. 주택담보대출 이자 상환액

### Part 4. 상황별 절세 전략 (8p)
- 18장. 맞벌이 부부 — 공제 배분 시뮬레이션 3가지 시나리오
- 19장. 중도입사/퇴사자 — "합산" vs "개별" 신고
- 20장. 프리랜서·N잡러 — 근로소득 + 사업소득 겹칠 때
- 21장. 외벌이 + 자녀 — 공제 몰빵 전략
- 22장. 1인 가구 — 축소되는 공제 보완책

### Part 5. 실전 체크리스트 & 워크시트 (6p)
- 23장. 페르소나별 공제 매트릭스 (4쪽 접지)
- 24장. 인쇄용 서류 체크 워크시트
- 25장. 제출 후 환급·추가 납부 대응 가이드
- 26장. 자주 묻는 질문 20선

### 부록 (4p)
- 세무서·국세청 홈택스 주요 URL/QR 모음
- 연도별 공제 한도 변화 비교표 (2022~2026)
- salarykorea.site 관련 계산기 한눈에 보기
